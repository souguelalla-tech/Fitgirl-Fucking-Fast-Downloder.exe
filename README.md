# FitGirl FuckingFast Link Extractor

## Description
Automated tool that extracts FuckingFast download links 
from FitGirl repacks pages and sends them directly to FDM 
or exports them to a text file. Requires Chrome or Brave browser.
- Chrome or Brave browser must be installed

---

## الوصف
أداة أتمتة تقوم بفتح صفحات موقع فيتغيرل تلقائياً،
تجمع جميع روابط "Fuckingfast" المقسمة على أجزاء،
ثم تخيرك بين:
- تصدير الروابط في ملف نصي
- إرسال الروابط مباشرة لبرنامج FDM

## المتطلبات / Requirements
- متصفح كروم أو برايف (أحدهما إلزامي)
